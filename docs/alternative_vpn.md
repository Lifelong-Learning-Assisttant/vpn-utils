Да, такие альтернативы есть. Наиболее подходящими под ваши критерии (Linux CLI, авторизация по токену, 3 ГБ+ бесплатного трафика и настройка DNS) в 2026 году являются следующие сервисы:
1. Hide.me (Лучший выбор для токенов)
Hide.me предлагает специализированный CLI-клиент для Linux, который изначально ориентирован на работу через токены.

    Токен: Команда hide.me token позволяет один раз авторизоваться и получить токен доступа, который затем используется для всех подключений без повторного ввода пароля.
    Трафик: Бесплатный план предоставляет 10 ГБ в месяц (на 2026 год данные могут варьироваться, но исторически это их стандарт).
    DNS: Поддерживает управление DNS через CLI, включая встроенную фильтрацию (SmartGuard) и возможность указания своих серверов. 

2. Windscribe
Windscribe имеет мощный CLI и очень гибкую систему авторизации.

    Токен: Вместо логина можно использовать Session Token или настроить конфигурацию через WireGuard (если у вас есть доступ к генерации конфигов), но основной CLI также поддерживает сохранение сессии.
    Трафик: 10 ГБ в месяц при подтверждении почты (или 2 ГБ без почты).
    DNS: В CLI есть команда windscribe dns, которая позволяет принудительно устанавливать DNS или использовать встроенный блокировщик R.O.B.E.R.T. 

3. Proton VPN (Новый CLI 2025-2026)
Proton обновил свой CLI в конце 2025 года, сделав его более современным. 

    Токен: Поддерживает авторизацию через OpenID/OAuth, что позволяет один раз авторизовать устройство.
    Трафик: Безлимитный (единственный крупный сервис с полностью бесплатным безлимитом).
    DNS: В версии 4.8.0+ для Linux появилась полноценная поддержка Custom DNS (команда protonvpn configure -> DNS Management), где можно прописать до 3 собственных IP-адресов DNS. 

4. Cloudflare WARP (через warp-cli)
Если вам не принципиальна смена страны (он просто шифрует трафик и дает IP Cloudflare), это самый быстрый вариант.

    Токен: Регистрация происходит один раз через warp-cli register, после чего создается постоянный ключ устройства.
    Трафик: Полностью безлимитный.
    DNS: Позволяет менять режим работы (например, warp-cli set-mode proxy или dns), чтобы заворачивать только DNS или только TCP.

Итог:

    Если важен именно процесс с токеном и 10 ГБ достаточно — выбирайте Hide.me.
    Если нужен безлимит и свой DNS — Proton VPN (через новый CLI).
    Если нужно 10 ГБ и гибкое управление — Windscribe.
